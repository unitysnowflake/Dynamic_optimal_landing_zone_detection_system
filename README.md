ЗОНА ПОСАДКИ, СОПОСТАВЛЕНИЕ, СПУСКАЕМЫЙ АППАРАТ, LiDAR, КАМЕРА, ПРЕПЯТСТВИЯ, КОРРЕКТИРОВКА

Система динамического определения зоны посадки спускаемого аппарата представляет собой автоматический модуль управления, основанный на данных с бортовых сенсоров, таких как LiDAR и оптическая камера. Основной целью данной системы является удержание спускаемого аппарата в заданном коридоре на управляемом этапе снижения и посадки, а также определения препятствий на поверхности, представляющих угрозу его опрокидывания и повреждения и увода на безопасный участок для посадки внутри коридора спуска. Такая система использует заранее загруженные орбитальные карты поверхности, одни из которых представляют собой визуальные, а другие цифровые карты высот. С помощью сопоставления снимков, полученных бортовой оптической камерой, а также сканов с LiDAR производится первичная оценка местности с учетом эффективности работы каждого из приборов. Данные о координатах с каждого модуля собираются и взвешиваются на основе параметров работоспособности сенсоров, получая таким образом конечные координаты для корректировки курса спускаемого аппарата с целью нахождения его в заданном коридоре спуска. Аппарат проходит минимум 2 таких корректировки курса прежде чем достигнет высоты для начала маневра уклонения от препятствий на поверхности перед касанием. Главным преимуществом такой системы, производящей совместный анализ данных с разных приборов, является более высокая устойчивость к изменениям внешних условий в районе посадки, таких как изменение освещенности из-за неподходящего времени суток или изменение видимости 6 ввиду пылевых бурь. Кроме того совместный анализ дает усредненное значение конечных координат, что дает статистическое преимущество в точности посадки в случае возможных искажений показаний одного из приборов.
